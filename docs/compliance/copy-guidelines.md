# 小程序内容合规文案指南（高风险规避）

> 目的：避免页面文案/结果呈现触发平台对“算命、抽签、星座运势、预测未来、封建迷信”等高风险内容的拒绝情形。

## 禁止（出现在页面/结果/引导文案）

页面与结果中应避免直接使用以下措辞（包含但不限于）：

- `算命`
- `抽签` / `抽签式` / `抽签结果`
- `星座运势` / `运势`
- `预测未来`
- `包你一定` / `必然发生`
- `宣扬邪教` / `邪教`
- `封建迷信` / `迷信`
- 任何“绝对化承诺/恐吓/强操控”的表述，例如“你一定会”“不做就会怎样”“保证改变命运”等。

## 推荐（可用于页面标题/结果呈现/按钮文案）

建议将功能定位为：

- `娱乐内容`
- `象征解读`（强调“象征”和“可能性”）
- `自我反思`
- `写作灵感/情境推演`

## 结果页免责声明（建议原文）

> 提示：本内容仅供娱乐与自我反思，不构成任何医疗、法律或投资建议。请理性对待，如需专业决策请咨询相关专业人士。

## “翻牌/呈现”命名规范（建议）

- 避免：`抽签`
- 使用：`翻阅/揭示/呈现牌面`

## 生成内容的安全约束（用于后端过滤与渲染）

- 以“象征启示、可能性、行为建议（非医疗/非承诺）”为主
- 禁止绝对化措辞与恐吓式内容
- 对生成文本与模板配置做敏感词/违规模式拦截
