# 塔罗象征解读（小程序 + 后端骨架）

本项目是一套以**象征解读、自我反思与写作灵感**为定位的互动体验（非预测、非算命）。你可以在微信小程序里选择主题与牌阵，通过「呈现牌面」获得文字启示，并可选写复盘、查看历史记录。

---

## 怎么玩

### 1. 今日灵感（首页）

- 从首页进入，先感受整体氛围。
- 你可以**选择牌阵**：进入选题与牌阵页，自行组合「关注主题」和「牌阵类型」。
- 也可以先点**开始呈现牌面**：直接进入解读流程（若未先选主题/牌阵，可按提示补选）。

历史入口在首页，方便回顾以往记录。

### 2. 选择牌阵

- **关注主题**：例如关系、事业、学习、自我成长等偏「反思方向」的分类（避免预测式说法）。
- **牌阵**：决定本次会呈现几个位置、每个位置代表什么叙事结构（如三张阵、十字阵、单张指引等）。
- 选好后点**开始呈现牌面**，进入解读页。

### 3. 象征解读（翻牌）

- 先点击**翻阅 / 布阵**类按钮，系统会为本次抽选牌面并对应到各位置（逻辑在后端或本地演示模式下完成）。
- 每张牌默认以**科技魔法风格的牌背**展示；**点击单张牌**可触发**翻面动画**，看到该张牌的符号与名称。
- **翻面之后**才会展开该位置对应的**象征解读文字**，便于你逐张消化。
- 牌面支持**正位 / 逆位**的呈现（视觉与解读会随位向变化）。
- 页内有**内容提示**（免责声明），请一并阅读：侧重娱乐与自我反思，不构成专业建议。

### 4. 复盘笔记（可选）

- 在解读页下方可以写下**复盘笔记**：例如你注意到了什么、打算做哪些小调整。
- 保存后，摘要会进入历史列表（具体是否上云取决于你是否接入后端与账号体系）。

### 5. 历史记录

- 按时间查看过往的**主题、牌阵与复盘摘要**，支持加载更多（若接口分页开启）。
- 可用于写日记式回顾，而非「预测记录」口吻。

### 6. 隐私与数据管理

- 在设置里查看**隐私说明**与数据相关提示。
- 若上线了真实后端，应支持按账号**删除个人相关数据**（以实际接入为准）。

## 使用心态（与产品定位）

- 把结果当作**隐喻、象征与灵感素材**，用于自我对话与小步行动，而不是「必然发生的事」。
- 避免将界面文案或分享语写成运势、算命、抽签式承诺，以利于平台合规与长期运营。
